# نظام نقطة شحن أبو عادل

نظام إدارة مالي وتشغيلي متكامل — تطبيق ويب احترافي يعمل **بدون إنترنت** بالكامل (Offline PWA).

## المميزات

- يعمل Offline بعد أول تشغيل (PWA + Service Worker)
- قاعدة بيانات محلية (LocalStorage) — بدون أي خدمات سحابية
- تثبيت على الهاتف (Add to Home Screen)
- نسخ احتياطي واستعادة JSON
- جاهز للتحويل إلى Android APK عبر Capacitor

## الأقسام

1. **لوحة التحكم** — مؤشرات مالية وتشغيلية مع فلترة بالفترة
2. **قسم الشحن** — تسجيل الأجهزة، وقت الدخول/الخروج، الحالات
3. **قسم المشروبات** — منتجات، سلة، فواتير، مخزون، أرباح
4. **إدارة الديون والزبائن** — كشف حساب PDF، تسديد، توزيع
5. **حسابات الشركاء** — أرصدة، سحوبات، تقسيم أرباح
6. **حصالة الشركاء** — خصم يومي تلقائي 10₪، مصروفات
7. **الصناديق** — 5 صناديق مستقلة مع سجلات
8. **الإدارة العامة** — مخازن، موردين، محصلون، تقارير
9. **الإعدادات** — مستخدمون، صلاحيات، قفل مديونية بكلمة مرور

## التشغيل (تطوير)

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

الناتج في مجلد `dist/` — يمكن رفعه على أي استضافة ثابتة أو تحويله لـ APK.

## التحويل إلى Android APK (Capacitor)

```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
npm run build
npx cap add android
npx cap sync
npx cap open android
```

ثم في Android Studio: Build → Generate Signed APK.

## النسخ الاحتياطي

- **تصدير:** قسم قاعدة البيانات → تصدير JSON
- **استيراد:** قسم قاعدة البيانات → استيراد JSON

## التقنيات

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- LocalStorage (قاعدة بيانات محلية)
- PWA (manifest + service worker)
- Capacitor (للتحويل إلى APK)
